# LNCT API
LNCT API wrapper, written in Python.

## Installation
Install using Pypi:
```bash
pip install lnctApi
```

## Usage
Example code to get Main Result:
```python
import lnctApi

student = lnctApi.accsoft(USER_ID, PASSWORD)
print(student.profile())
```
